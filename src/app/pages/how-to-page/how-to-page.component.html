Getting started with Partea

A step-by-step guide how-to use Partea

# For project coordinators

1. Create a new project

Go to the Projects page, type in the project name and click the create button.

For this you need to be logged in. Please register or log in first.

2. Initialize the project parameters

In the Parameters section of the project page, define the Method (Univariate Analysis or Survival Regression) and further global parameters.

This information helps other participants to preprocess their data the right way.

3. Invite collaborators

In the Collaborators section, create tokens and share them with directly with your partners as an invitation to your study. You can also share the master token for larger studies that you can share multiple times.

4. Run project

In the Collaborators section of the project page, press the Run Project button when every participant has joined. If you also want to join as a participant, create a token for yourself and follow the steps in the For project participants section.

5. Enjoy the results :-)

In the Summary section of the project page, you get an overview about your project. In the Results section of the proejct page, you can see the plots of your models and a table containing the data. You can download all the results directly from this page.

# For project participants

1. Download the PARTEA client

Download the PARTEA GUI client:

  • Linux GUI
  • macOS GUI
  • Windows GUI (Coming soon)

Google Chrome needs to be installed


Starting the GUI can take some time

Advanced users can also use the Docker CLI tool:

  • docker pull registry.blitzhub.io/partea-cli

Download Sample Data

  • Rossi - 3 Clients (Source)
  • Duration Column: week
  • Event Column: arrest

2. Login and join project

Start the client and join the project with your credentials (Sign up) and project token you received from the coordinator.

If you received a master token, you don't necessarily need an Account. You can only enter a username and the master token.

Be aware, that you don't have access to the results if you do not join with an account.

3. Run client

Specify your file columns and choose the path to your local file.

After everything specified, run the local client by pressing the Run button.

4. Wait for results

Lay back, the computation will start automatically after all participants have joined.

Be aware that your client needs to keep running. Otherwise the computation cannot finish succesfully.

5. Enjoy the results :-)

Go to the PARTEA webapp, login with your account and select the corresponding project.

In the Results section of the proejct page, you can see the plots of your models and a table containing the data. You can download all the results directly from this page.
